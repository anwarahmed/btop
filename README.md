# Btop

My Btop setup

https://github.com/aristocratos/btop

## Files

Copy the contents of this repository to `~/.config/btop/`.
The following files can be ignored:

- `README.md`
- `.git`
- `.gitignore`

## Installation

### Homebrew

```sh
brew install btop
```

### Arch Linux

```sh
sudo pacman -S btop

```
